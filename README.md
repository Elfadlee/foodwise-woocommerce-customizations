Foodwise – WooCommerce Customizations

This repository contains custom frontend and WooCommerce-related modifications
implemented for the Foodwise e-commerce website.

Implemented Features
- Displaying the “Items on Offer” section at the top
- Adding product category filters
- Improving the add-to-basket user experience
- Mobile layout and spacing adjustments
- WooCommerce result count & sorting behavior fixes

Technologies
- WordPress
- WooCommerce
- Custom CSS
- Custom JavaScript
- PHP (WooCommerce hooks & snippets)

Live Website
https://foodwisekw.com

Notes
